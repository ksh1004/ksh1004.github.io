## 핵심 개념

### 정수형(int)

Python에서 소수점 없는 숫자는 정수형(`int`)으로 저장된다. 정수형 변수끼리는 사칙연산이 그대로 적용된다.

```python
a = 5
b = 10
```

여기서 `a`, `b`는 모두 `int` 타입 변수다.

### 산술 연산자 `-`

두 정수를 뺄셈하면 그 결과도 정수형이다. 순서가 중요하며 `a - b`와 `b - a`는 부호가 반대다.

```python
a - b  # 5 - 10
```

---

## 접근 방식

1. `a = 5`, `b = 10` — 두 변수에 정수 저장
2. `a - b` 계산 → `5 - 10 = -5`
3. `print(a - b)` → 계산 결과인 `-5`를 출력

뺄셈에서 작은 수에서 큰 수를 빼면 음수가 나온다는 점만 확인하면 되는 문제다.

---

## 풀이 코드

```python
a = 5
b = 10
# a - b: 5에서 10을 빼므로 결과는 음수
print(a - b)
```

**출력 결과:**

```
-5
```

---

## 시간 / 공간 복잡도

| 구분 | 복잡도 | 이유 |
|---|---|---|
| 시간 | O(1) | 변수 2개에 대한 단일 뺄셈 연산 |
| 공간 | O(1) | 변수 2개만 사용, 입력 크기와 무관 |

---

## 노하우 & 주의사항

- `a - b`와 `b - a`를 헷갈리지 않도록 항상 **코드에 적힌 순서 그대로** 계산한다.
- 결과가 음수여도 정수형(`int`)이라는 사실은 변하지 않는다 — Python은 음수 정수도 `int`로 취급한다.
- 비슷한 유형의 문제에서 `+`, `*`, `/`, `//`(몫), `%`(나머지)로 바뀌어 나올 수 있으니, 연산자별 결과 차이를 함께 알아두면 좋다.
