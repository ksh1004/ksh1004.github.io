## 핵심 개념

### 변수(Variable)란

**데이터를 저장하고 조작할 수 있는 메모리 공간의 이름**이다.

프로그램이 실행되면 컴퓨터 메모리(RAM) 어딘가에 값이 저장되는데, 그 위치를 사람이 알아보기 쉬운 이름으로 부르는 것이 변수다. 메모리 주소를 직접 다루지 않고 이름만으로 값을 읽고 쓸 수 있게 해준다.

```python
age = 20        # age라는 이름의 공간에 20을 저장
name = "Kim"     # name이라는 이름의 공간에 "Kim"을 저장
```

### 변수의 3요소

| 요소 | 설명 |
|---|---|
| 이름 | 값을 참조하기 위한 식별자 (`age`, `name`) |
| 값 | 실제로 저장된 데이터 (`20`, `"Kim"`) |
| 타입 | 저장된 데이터의 종류 (정수, 문자열 등) — Python은 자동으로 추론 |

---

## 접근 방식

"변수란 무엇인가"를 묻는 개념 문제는 정의를 정확히 아는지가 핵심이다.

1. 변수는 값 자체가 아니라 **값이 저장된 공간을 가리키는 이름**이다.
2. 같은 이름이라도 대입(`=`)을 다시 하면 저장된 값이 바뀐다 — 이름은 고정, 값은 변경 가능.
3. "메모리 공간의 이름"이라는 표현이 핵심 키워드다.

---

## 풀이 코드

```python
# 변수 선언: age라는 이름으로 메모리 공간을 만들고 20을 저장
age = 20
print(age)  # 20

# 같은 변수에 다른 값을 대입하면 저장된 값만 바뀐다
age = 21
print(age)  # 21
```

**출력 결과:**

```
20
21
```

## 시간 / 공간 복잡도

| 구분 | 복잡도 | 이유 |
|---|---|---|
| 시간 | O(1) | 변수 대입·조회는 상수 시간 연산 |
| 공간 | O(1) | 변수 1개당 고정된 메모리 공간 사용 |

## 노하우 & 주의사항

- 변수는 "상자"에 비유되곤 하지만, 정확히는 **메모리 주소에 붙인 이름표**에 가깝다.
- Python은 변수 선언 시 타입을 명시하지 않아도 되는 **동적 타이핑** 언어다. `age = 20` 이후 `age = "스무살"`처럼 다른 타입 값도 재대입 가능하다.
- 변수명은 숫자로 시작할 수 없고, 예약어(`print`, `if` 등)는 변수명으로 쓸 수 없다.
