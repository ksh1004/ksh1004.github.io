## 핵심 개념

### 몫 연산자 `//`

`a // b`는 `a`를 `b`로 나눈 **몫**을 정수로 반환한다. 일반 나눗셈(`/`)과 달리 소수점 이하를 버린다.

```python
35 / 8   # 4.375 (실수)
35 // 8  # 4      (정수 몫만)
```

### 나머지 연산자 `%`

`a % b`는 `a`를 `b`로 나눈 **나머지**를 반환한다.

```python
35 % 8   # 3
```

몫과 나머지는 항상 세트로 이해하면 편하다: `a = (a // b) * b + (a % b)`가 항상 성립한다.

---

## 접근 방식

1. `a = 5`, `b = 30` → `c = a + b = 35`
2. `35 // 8` 계산 → 8×4=32, 8×5=40이므로 몫은 `4`
3. `35 % 8` 계산 → 35 - 32 = 나머지 `3`
4. 몫을 먼저, 나머지를 나중에 출력 (문제에서 요구한 순서)

---

## 풀이 코드

```python
a, b = 5, 30
c = a + b       # c = 35

print(c // 8)   # 몫: 35 // 8 = 4
print(c % 8)    # 나머지: 35 % 8 = 3
```

**출력 결과:**

```
4
3
```

| 구분 | 복잡도 | 이유 |
|---|---|---|

| 공간 | O(1) | 변수 3개만 사용, 입력 크기와 무관 |

## 노하우 & 주의사항

- 출력 순서(몫 → 나머지)가 문제에서 명시되어 있으므로, 코드 작성 시 `print()` 호출 순서를 반드시 문제 요구사항과 맞춰야 한다.
- 검산 팁: `(몫 × 나누는 수) + 나머지 = 원래 값`이 성립하는지 확인하면 실수를 잡을 수 있다. (`4 × 8 + 3 = 35` ✓)
- `/`(실수 나눗셈)와 `//`(정수 몫)를 혼동하지 않도록 주의. `35 / 8`은 `4.375`로, 이 문제가 원하는 정수 몫과는 다르다.
