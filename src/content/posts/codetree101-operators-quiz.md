## 핵심 개념

### 나머지 연산자 `%`

`a % b`는 `a`를 `b`로 나눈 **나머지**를 반환한다.

```python
15 % 7  # 15 ÷ 7 = 2 나머지 1 → 결과는 1
```

### 복합 대입 연산자 `%=`

`c %= 7`은 `c = c % 7`을 줄여 쓴 표현이다. Python의 복합 대입 연산자는 "기존 값에 연산을 적용한 뒤 다시 저장"하는 패턴을 짧게 쓴 것뿐이다.

| 축약형 | 원래 표현 |
|---|---|
| `c += 3` | `c = c + 3` |
| `c -= 3` | `c = c - 3` |
| `c *= 3` | `c = c * 3` |
| `c %= 3` | `c = c % 3` |

---

## 접근 방식

1. `a = 5`, `b = 10`
2. `c = a + b` → `c = 15`
3. `c %= 7` → `c = 15 % 7` → `c = 1` (15를 7로 나누면 몫 2, 나머지 1)
4. `print(c)` → `1` 출력

핵심은 `%=`를 보고 "나머지를 구해서 다시 저장한다"로 바로 풀어 쓸 수 있는지다.

---

## 풀이 코드

```python
a = 5
b = 10
c = a + b   # c = 15

c %= 7      # c = c % 7 = 15 % 7 = 1
print(c)
```

**출력 결과:**

```
1
```

---

## 시간 / 공간 복잡도

| 구분 | 복잡도 | 이유 |
|---|---|---|
| 시간 | O(1) | 고정된 횟수의 산술 연산 |
| 공간 | O(1) | 변수 3개만 사용, 입력 크기와 무관 |

---

## 노하우 & 주의사항

- `%=`를 `=`(단순 대입)로 착각해서 `c = 7`로 잘못 계산하는 실수가 흔하다. 복합 대입 연산자를 보면 항상 원래 형태(`c = c % 7`)로 먼저 풀어 쓰고 계산하는 습관을 들이자.
- 나머지 연산 결과는 나누는 수(여기선 7)보다 항상 작다 — 검산할 때 이 성질로 답이 맞는지 빠르게 확인할 수 있다.
- 비슷한 유형에서 `//=`(몫 재대입)와 `%=`(나머지 재대입)를 혼동하지 않도록 주의한다.
